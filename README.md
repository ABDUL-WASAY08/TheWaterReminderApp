## 📋 About

**Water Reminder** is a Flutter-based mobile application designed to help you stay hydrated throughout the day. Set customized reminders at specific times — just like an alarm — and the app will notify you when it's time to drink water.

Built with love by [Abdul Wasay](https://github.com/ABDUL-WASAY08) as a learning project to master Flutter development.

---

## ✨ Features

- ⏰ **Custom Alarm-Style Reminders** — Set specific times to be reminded to drink water
- 🎨 **Beautiful Purple-Themed UI** — Clean and modern gradient interface
- 🔔 **Push Notifications** — Get notified even when the app is closed
- 📊 **Water Intake Tracking** (coming soon)
- 📅 **Daily/Weekly Schedule** (coming soon)
- 📈 **Hydration Stats** (coming soon)

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (3.11.5 or higher)
- [Dart SDK](https://dart.dev/get-dart) (included with Flutter)

### Installation

```bash
# Clone the repository
git clone https://github.com/ABDUL-WASAY08/TheWaterReminderApp.git

# Navigate to the project directory
cd TheWaterReminderApp

# Get dependencies
flutter pub get

# Run the app
flutter run
```

### Build for Release

```bash
# Android APK
flutter build apk

# iOS (macOS only)
flutter build ios

# Web
flutter build web

# Windows
flutter build windows
```
